# 1. 프로그램 설계서 (5점)
학원에서 쓸 수 있는 출석 프로그램
## 1. 프로젝트 주제
(예: 학생 성적 관리 프로그램)
학생 출석 프로그램
## 2. 핵심 기능 목록
(구현할 기능들을 순서대로 나열하세요.)
1.  출석 체크 : 학생 이름을 입력하면 프로그램이 해당 이름을 출석 명단에 추가하고 “출석 완료” 메시지를 띄워줍니다.
2.  출석 명단 보기 : 지금까지 출석한 학생들의 목록을 번호와 함께 보여줍니다.
3.  프로그램 종료 : 프로그램을 종료합니다. (사용자가 직접 종료 선택 가능)
4.  관리자 비밀번호 : 관리자가 아닌 사람은 출석 명단을 못 보게 잠금 기능 추가

## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)
count , choice 
**방법 1: 배열 사용 (기본)**
```c
// 예시: 학생 5명의 이름과 성적 저장
char names[5][20];
int scores[5];
int count = 0;  // 현재 학생 수
```
#define MAX_STUDENTS 100   // 최대 학생 수
#define NAME_LENGTH 30     // 이름 최대 길이
**방법 2: 구조체 사용 (선택)**
```c
// 예시: 학생 정보를 구조체로 묶어서 관리
struct Student {
    char name[20];
    int kor;
    int eng;
    int math;
    float avg;
};
struct Student students[5];  // 학생 5명 저장
```
